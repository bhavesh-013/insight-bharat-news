# 📰 Insight Bharat – Premium News Application

> **Truth Beyond Headlines**

Insight Bharat is a premium editorial-style news application inspired by traditional newspapers and powered by **NewsAPI**. It delivers live headlines, category-based news, keyword search, bookmarking, and a responsive newspaper reading experience.

---

## ✨ Features

### 🏠 Home Page

* Dynamic hero section with top headlines
* Breaking News ticker
* Editor's Picks
* Premium editorial newspaper layout
* Responsive design

### 📰 Categories

Browse news by category:

* World
* Politics
* Business
* Technology
* Sports
* Entertainment
* Science
* Health
* Opinion

### 🔍 Search

* Search news by keyword
* Live NewsAPI integration
* Search results page
* Loading and error states

### 📖 Article Details

* Full article view
* Featured image
* Source & author information
* Published date
* Estimated reading time
* Read original article

### 🔖 Bookmarks

* Save articles for later
* LocalStorage support
* Quick access to saved news

### 🌙 Night Edition

* Elegant dark editorial theme
* Newspaper-inspired typography
* Smooth theme switching

# 🌐 API

This project uses **NewsAPI** to fetch live news.

Example endpoints:

```
Top Headlines
https://newsapi.org/v2/top-headlines

Search
https://newsapi.org/v2/everything
```

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/insight-bharat.git
```

### Open Project

```bash
cd insight-bharat
```

### Run

Open with **Live Server** in VS Code or visit:

```
http://127.0.0.1:5500
```
